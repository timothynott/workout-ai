## Database & ORM

<!-- ~80 words: Neon Postgres with Drizzle ORM. Neon was already required for auth. Drizzle over Prisma because Drizzle is edge-native — no binary query engine, no code gen step, small bundle. Neon's branch-per-environment strategy means preview deploys never touch production data. -->
